# Domain-Driven Design in Laravel

## Installation

- Clone repository:
    ```batch
    git clone git@github.com:Mexidense/ddd-laravel-skeleton.git
    ```

- Install packages:
    ```batch
    composer install  
    ```

- Generate & set application key
    ```batch
    php artisan key:generate
    ```

## Usage

- Run server
    ```batch
    php artisan serve
    ```
